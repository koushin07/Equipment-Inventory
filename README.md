

# Equipment Inventory

## Description

 A Regional Inventory System capable of facilitating communication between municipalities both within and outside of the province/s
 

## Getting Started

### Prerequisites

- [Java 11](https://www.java.com/)
- [Maven](https://maven.apache.org/)

### Installing

Run the application

```bash
  $ mvn spring-boot:run 
```
